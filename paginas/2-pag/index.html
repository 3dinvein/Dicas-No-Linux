FERRAMENTA SMART PATCH TOOl NO PROGRAMA KRITA

A Smart Patch Tool é uma ferramenta encontrada no programa Krita. O que ela faz é simplesmente copiar uma área selecionada e colar em uma outra área que a gente queira e que esteja faltando, ela cola de forma inteligente, fazendo com que as cores das duas imagens se combinem para evitar diferenças.
