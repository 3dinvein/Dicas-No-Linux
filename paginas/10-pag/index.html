COMANDO LSHW

COMO INSTALAR O LSHW


sudo apt install lshw

COMANDO PARA LISTAR NOSSO HARDWARE


sudo lshw -short

Decidi fazer este vídeo pois o comando que utilizei nessa semana é muito bom, ele permite que vejamos os componentes do nosso hardware. Claro que esse vídeo serve mais para quem lida com servidores sem interface gráfica, e que precisam saber o tipo de componentes o servidor possui. Veja bem, não estou falando apenas de tamanho mas sim do tipo deles. Por exemplo, se a memória é DDR 2 ou 3. Então esse comando nos ajuda ver o nome dos componentes, o tipo, a quantidade e os slots ocupados, e também se são dual channel ou não. Esse comando é ótimo para que não seja necessário ter que abrir o gabinete só para ver o nome ou o tipo do componente.
